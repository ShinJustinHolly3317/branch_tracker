持有約 15–20 交易日 只碰有量的熱門股 避開三根大紅棒 🟡 驗證員：部分通過（資料尚不足 1 年）

這策略在幹嘛？

大盤熱的時候，很多股票會「慢慢變強」而不是一次噴完。這套做法專找溫和上漲、成交量夠大、且分點（券商）有在買的標的，抱大約 3～4 週，不是今天買後天賣。

白話：跟著「有資金在顧、但還沒被炒爛」的動能股，用小部位參與熱門題材。

你的三個在意點 → 具體規則

❌ 不要短進短出

實盤計畫持有 15–20 個交易日（約 3–4 週）。只有跌破停損（例如 -8%）或動能明顯消失才出場。

❌ 不要交易量太小

分點買賣合計至少 12 萬股，確保進出場不會卡死。大型股像台積電、金融股通常較容易過關。

❌ 不要已經漲太多的

近 3 日單日漲幅 ≤ 3%、三日合計 ≤ 7%；且今天漲幅 ≤ 2%。動能區間 2.5%–7%（3 日），避開連續三根大紅棒。

  • 近 3 日任一日漲幅 ≤ 3%，三日加總 ≤ 7%；進場日漲幅 ≤ 2%
  • 當日分點買賣量 ≥ 12 萬股
  • 分點淨買超占成交量 ≥ 3%
  • 最大買超分點占所有分點淨買賣絕對值 ≥ 20%（有主力痕跡）
  • 時間出場：持有滿 15–20 交易日，無論盈虧先減碼或全出
  • 停損：自進場價下跌 8% 出場
  • 停利：獲利達 15% 可先賣一半，其餘用移動停利（例如從高點回撤 5%）
  • 動能失效：若 3 日動能轉負且分點轉淨賣，提前出場
  • 總資金只用 10%–20% 做這套動能策略（你說的小部分錢）
  • 同時持有 3～5 檔，每檔約 3%–5% 總資金
  • 避免 all-in 單一熱門股；0050 / 2330 可當「底倉」但分點條件常較難過
